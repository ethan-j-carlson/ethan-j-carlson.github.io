IPA Chart Directory

/aɪ̯.pʰi.eɪ̯ t͡ʃɑɹt daɪ̯.ˈɹɛk.tə.ɹi/

This page contains an International Phonetic Alphabet chart (IPA chart), which is a mostly standard way to organize all of the IPA's symbols based on their phonological features. Wikipedia has a thorough example, and ipachart.com provides one with audio for each symbol. The colored symbols link to the page about the symbol's phonetics, name, encoding and more.
Per the standard, voiced consonants and rounded vowels are to the right in the pair.
